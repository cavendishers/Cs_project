1. 插件简介

本插件为 Chrome 浏览器扩展（Extension），通过加载本项目目录中的 manifest.json 完成注册与运行。插件通常会在以下位置体现：

浏览器右上角扩展栏出现插件图标（来自 icons/）

在指定网页中注入脚本/样式（通常来自 scripts/、styles/）

使用通用工具方法（来自 utils/）

2. 目录结构说明（对应你的项目）

manifest.json
插件入口配置文件（必须）。定义插件名称、版本、权限、脚本注入规则、popup 页面、后台脚本等。

icons/
插件图标资源目录。包含 16/32/128 尺寸的 PNG。

scripts/
插件核心脚本目录。

styles/
插件样式目录。

utils/
通用工具函数目录.

参考文件/
辅助资料/说明/原型/截图/测试数据等，不参与插件运行（除非在 manifest 中被引用）。

3. 安装方式（本地加载 / 开发调试）
3.1 准备

确保你已经把项目代码下载到本地（包含 manifest.json 的那一层目录）。

使用 Chrome 打开扩展管理页：
在地址栏输入：chrome://extensions/

3.2 开启开发者模式

右上角打开 「开发者模式」

3.3 加载已解压的扩展程序

点击 「加载已解压的扩展程序」

选择你的项目根目录（必须能看到 manifest.json 的那一层）

加载成功后会在扩展列表中看到你的插件

3.4 刷新目标页面，并开嗦

3.4 固定到工具栏（推荐）

点击浏览器右上角“拼图”图标 → 找到插件 → 点击“图钉”固定
