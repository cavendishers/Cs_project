# 浏览器插件（Chrome Extension）

## 插件简介
本插件为 **Chrome 浏览器扩展（Extension）**，通过加载本项目目录中的 `manifest.json` 完成注册与运行。插件通常会在以下位置体现：

- 浏览器右上角扩展栏出现插件图标（来自 `icons/`）
- 在指定网页中注入脚本/样式（通常来自 `scripts/`、`styles/`）
- 使用通用工具方法（来自 `utils/`）

---

## 目录结构说明
- `manifest.json`：插件入口配置文件（必须），定义名称、版本、权限、注入规则、popup、后台脚本等  
- `icons/`：插件图标资源目录（常见包含 16/32/128 等尺寸 PNG）  
- `scripts/`：插件核心脚本目录  
- `styles/`：插件样式目录  
- `utils/`：通用工具函数目录  
- `参考文件/`：辅助资料/说明/原型/截图/测试数据等（默认不参与插件运行，除非在 `manifest.json` 中被引用）

---

## 安装方式（本地加载 / 开发调试）

### 1) 准备
- 确保已将项目代码下载到本地，并且**选择目录时能看到 `manifest.json`**
- 打开 Chrome 扩展管理页：在地址栏输入  
  `chrome://extensions/`

### 2) 开启开发者模式
- 在页面右上角打开 **「开发者模式」**

### 3) 加载已解压的扩展程序
- 点击 **「加载已解压的扩展程序」**
- 选择项目根目录（**必须包含 `manifest.json`**）
- 加载成功后会在扩展列表中看到你的插件

### 4) 刷新目标页面，开嗦
- 打开/刷新目标网页（必要时 `Ctrl + F5` 强刷），插件即可生效

### 5) 固定到工具栏（推荐）
- 点击浏览器右上角 **“拼图”** 图标
- 找到插件 → 点击 **“图钉”** 固定到工具栏
